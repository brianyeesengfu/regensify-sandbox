# REGENSIFY — Neo Botanical Homepage Sandbox

A full homepage redesign mockup for [regensify.com](https://www.regensify.com) using the **Neo Botanical** design direction — warm terracotta and sand palette, Playfair Display serif headlines, category grid navigation, and a nature-meets-science wellness aesthetic.

## Live sandbox

This is hosted via GitHub Pages. View it at:
**https://[your-username].github.io/regensify-sandbox/**

## What's inside

- Sticky navigation with category filtering
- Promo bar with SG10OFF code
- Split hero with product image and badge
- Trust strip (clinical actives, clean formula, Made in SG, cruelty-free)
- Interactive category grid (filters product listings live)
- Full product grid with real Shopify product images and prices
- Bestseller feature banner (Hair Melanin Booster Lotion)
- 4-step haircare system explainer
- Science ingredient cards (Redensyl, Greyverse™, Capixyl™, AnaGain™ Nu)
- Bundle deals section
- Clinic services section (Regenera Activa, PLINEST Hair)
- Testimonial section
- Newsletter signup
- Full footer with social links and payment icons

## Design tokens

| Token | Value |
|---|---|
| Primary font | Playfair Display (serif, headlines) |
| Body font | Inter (sans-serif) |
| Clay (primary accent) | `#8B4513` |
| Sand (background) | `#faf5ec` |
| Bark (text) | `#2c1a0e` |
| Moss (save badges) | `#3d5a3e` |

## Deploy to GitHub Pages

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your sandbox URL will be: `https://[username].github.io/regensify-sandbox/`
